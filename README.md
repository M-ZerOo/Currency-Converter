# Currency-Converter


Introduction
------------
Currency Converter is an app that converts your desired amount from one currency to another using [FixerApi](https://apilayer.com/marketplace/fixer-api).

Technologies
------------
### Language:
- Kotlin
- XML
### Architecture patterns:
- MVVM
### Libraries:
- Room
- RxJava
- Retrofit
- Gson Converter
- OkHttp
- Hilt
- ViewModel
- LiveData
